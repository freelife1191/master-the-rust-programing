# Rust의 산술 연산자

| Operator   | Symbol | Description                                |
|------------|--------|--------------------------------------------|
| Addition   | `+`      | 두 숫자를 더합니다                |
| Subtraction| `-`      | 첫 번째 숫자에서 두 번째 숫자를 뺍니다   |
| Multiplication | `*`  | 두 숫자를 곱합니다                     |
| Division   | `/`      | 첫 번째 숫자를 두 번째 숫자로 나누고 정수의 경우 0을 향해 자릅니다 |
| Modulus    | `%`      | 첫 번째 숫자를 두 번째 숫자로 나눈 나머지를 계산합니다    |
